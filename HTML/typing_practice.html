Beginner Practice Test

Lets start out by typing this quote into the text box as practice. Typing is quite easy and enjoyable when you have learned to do it correctly. It is amazing just how quickly you can be when you learn to touch type all the letters exactly, from A to Z. Just relax and realize that typing with excellent accuracy is far better than typing fast; you will find it is quicker in the end.

Practice the above quote until you feel comfortable with your keyboard layout. Aim to make minimal mistakes, then proceed to the next exercise. At the average typing speed of 40 WPM, it would take one minute and 34 seconds to type this text.

Intermediate Typing Test

Capital letters are used frequently - learning to type them efficiently is "key" to excellent typing! QWERTY keyboards have two large size SHIFT keys, which make it very convenient. Just to make sure to utilize the opposite hand to hold the shift key when you type a capital letter. It's much easier doing it like this.

Numbers Practice Test

Being able to type numbers is just as important. While it can be tricky, it will allow you to be more diverse, allowing you to have a wider skill set when typing. Typing numbers might be difficult for some, simply for the fact that some keyboard layouts don't have the Numpad on the side. But mastering the number row keys is crucial.

Final Practice Test

Getting through this final practice test is a great hurdle in your quest of becoming a Type-Master. This practice test has a combination of numbers and symbols. How fast can you type? Typing @ > 40 WPM (normal text) is considered fast, but "pros" often exceed ~80WPM! Words Per Minute, or "WPM" = [((Total Characters)/5) * (60/Seconds)] @ 100% accuracy. ***However, Adjusted WPM = [Chars. - (<# of Errors>)*5) * (60/Seconds)***.
